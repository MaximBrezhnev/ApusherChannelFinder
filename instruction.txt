* Для получения ключа (developerKey) для парсинга Ютуба необходимо следовать инструкции из ссылки: https://blog.hubspot.com/website/how-to-get-youtube-api-key
* Для получения строки сессии для парсинга Телеграма необходимо:
1. Получить api_id и api_hash согласно инструкции: https://core.telegram.org/api/obtaining_api_id
2. Запустить файл get_session_string.py, введя нужные данные в консоль: результатом работы будет необходимая строка сессии

Примечание: подобная реализация авторизации для Телеграма обусловлена необходимость распространения приложения в качестве exe-файла
